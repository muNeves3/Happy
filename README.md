## :question: About the project
The app made in the third edition of Next-Level-Week, offered by rocketseat, aims to make connections between orphanages and people who want to visit them

## :rocket: Technologies 

### FrontEnd/Web:
  - Typescript
  - ReactJS
  - React-router-dom
  - Styled-components
  - React-Switch
  - Leaflet
  - Mapbox API
  - Polished

### Backend
  - Typescript
  - NodeJS
  - Express
  - SQlite
  - TypeORM
  - Multer
  - Yup
  - Express Async Erros
  - Cors
